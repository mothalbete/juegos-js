🎮 Juegos JS
Colección de mini‑juegos desarrollados con JavaScript, HTML y CSS, creados para practicar lógica, interacción con el DOM y diseño web.
Todos los juegos funcionan directamente en el navegador y no requieren instalación.

📁 Estructura del proyecto
juegos-js/
│── index.html        # Página principal con el menú de juegos
│── juegos.css        # Estilos generales
│── juegos/           # Carpeta con los juegos individuales
│── img/              # Recursos gráficos

🕹️ Juegos incluidos
-Adivina el número
-Piedra, papel, tijera
-Conversor de monedas
-Trivia
-To-do List

🚀 Cómo ejecutar
No requiere instalación ni dependencias.

1-Clona el repositorio: git clone https://github.com/mothalbete/juegos-js
2-Abre el archivo index.html en tu navegador.
3-Selecciona un juego desde el menú principal.

🛠️ Tecnologías utilizadas
-HTML5
-CSS3
-JavaScript
